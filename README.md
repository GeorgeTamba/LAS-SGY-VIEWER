# LAS & SGY VEIWER (Petrophysics & Seismic Workstation)

A high-performance, web-based interactive workstation for visualizing well logs (.las) and 3D/2D seismic data (.sgy). Built entirely in Python using Streamlit.

## Features

- **Multi-Track Well Logs:** Synchronized cross-track zooming and custom color palettes.
- **Interactive 3D Wellbore:** Generates dynamic 3D tube models based on curve data.
- **Seismic Vector Engine:** Renders massive 3D Inlines and Crosslines with dynamic amplitude clipping.
- **Incredible Mode Tooltips:** Maps raw machine traces directly to real-world geospatial coordinates.
